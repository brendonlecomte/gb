# GB Emulator (C)

## Goal
To build a clone of the GB/GBC. From C based Emulator right through to custom
hardware that fits inside the original gameboy case.

## Steps
- [ ] C based emulator
- [ ] Port to uC
- [ ] Custom HW
- [ ] Fully portable clone


## Building
To build and run unittests against the code
'''
make
'''
